# AI Insurance Advisor

React + Vite + TypeScript dashboard UI for exploring insurance-style planning views. This is a **demo / educational** interface, not financial or legal advice.

## Scripts

- `npm run dev` — start dev server (port 8080)
- `npm run build` — production build
- `npm run preview` — preview production build
- `npm run lint` — ESLint
- `npm run test` — Vitest unit tests

## Disclaimer

Content and comparisons shown in the app are illustrative only. Consult a licensed professional for real insurance or investment decisions.
